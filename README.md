# Diablo

웹 기반으로 디아블로 스타일의 핵심 재미만 빠르게 구현하는 액션 RPG 프로토타입입니다.

## 목표

- 핵심 루프: 이동, 몬스터 처치, 자동 스킬 발동, 전투 몰입감
- 조작 단순화: 플레이어는 이동만 직접 조작
- 스킬은 각 쿨타임이 돌면 자동 시전
- 짧은 세션에서도 전투 손맛이 느껴지도록 구성

## 구현 방향

- 기본은 2D 탑다운 액션
- 필요하면 `Three.js` 기반으로 3D 캐릭터/몬스터/맵을 붙여 확장
- 3D 사용 시 `glb` 에셋을 활용해 동일한 게임 루프 유지

## 핵심 시스템

- 플레이어 이동
- 적 스폰 및 추적
- 자동 전투 스킬
- 스킬별 쿨타임
- 피격, 체력, 사망 처리
- 드랍 또는 경험치 기반 성장 요소

## 개발 원칙

- 먼저 재미있는 전투 루프를 만든다
- UI, 연출, 콘텐츠 양보다 조작감과 타격감을 우선한다
- 구조는 웹에서 빠르게 반복 개발 가능한 형태로 유지한다
